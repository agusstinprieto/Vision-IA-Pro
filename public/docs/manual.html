Guía rápida para Guardias de Caseta y Gerentes de Flota. Cómo supervisar el sistema autónomo VISION PRO. Quick guide for Gate Guards and Fleet Managers. How to supervise the autonomous VISION PRO system.

01 Guardia / Monitorista Guard / Operator

1

Command Center (Torre de Control)

PUNTO DE INICIO

Seleccione una unidad en el mapa o lista. El sistema vinculará automáticamente al chofer asignado para iniciar cualquier protocolo de IA. Select a unit from the map or list. The system will automatically link the assigned driver to start any AI protocol.

2

Monitoreo IA (Fatiga)

SURVEILLANCE EN VIVO

Active el streaming para detectar microsueños, bostezos y niveles de estrés. Las alertas se registrarán en el log central para acción inmediata. Activate streaming to detect microsleeps, yawns, and stress levels. Alerts will be logged centrally for immediate action.

3

Auditoría IA (Gemini)

REVISIÓN DE INTEGRIDAD

Use este botón para una inspección profunda: busca celulares, cigarros, acompañantes o malestar físico. Genera un reporte forense detallado. Use this button for deep inspection: looks for cellphones, cigarettes, companions, or physical distress. Generates a detailed forensic report.

  • Registro biométrico histórico

Análisis robótico profundo. Utiliza Google Gemini para auditar la cabina. No solo mira, entiende comportamientos complejos: celulares, cigarros, acompañantes o manipulación de carga.

  • Análisis forense de cabina
  • Detección de objetos y anomalías
  • Reporte ejecutivo automático

Automatización Híbrida IA

1. Monitoreo Pasivo

La IA Biométrica vigila 24/7 sin intervención humana, enviando alertas de fatiga al monitor central.

2. Disparo por Evento

Si el Monitoreo detecta "Ojos Cerrados" o "Microsueño", se ACTIVA automáticamente la Auditoría IA (Gemini).

3. Auditorías Programadas

Usted puede pre-configurar auditorías profundas cada 4 horas o en cruces de zonas de alto riesgo.

Solución de Problemas Troubleshooting

Problema

"Cámara Desconectada" / "Sin Señal"

Verifique que el cable de red del Arco no esté dañado. Reinicie el switch PoE.

Problema

No lee las placas de noche

Limpie el cristal frontal de la cámara IR. Verifique que los focos LED infrarrojos enciendan.
